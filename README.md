Overview of the SpendWise Dashboard

The SpendWise Dashboard is a simple and responsive financial management web page built using HTML and styled with CSS. Its purpose is to provide users with a clear overview of their spending across different expense categories in an organized and visually appealing layout.

The dashboard consists of three main sections:

Sidebar Navigation: Located on the left side, it contains links to key sections such as Dashboard, Expenses, Reports, Budget, and Settings, allowing users to navigate the application easily.
Header: Displays the title "Financial Dashboard" along with a personalized welcome message for the user, making the interface more user-friendly.
Main Content Area: Shows expense summary cards for categories including Food, Transport, Rent, Entertainment, Savings, and Utilities. Each card displays the total amount spent in that category, making it easy to monitor financial activity at a glance.

The dashboard uses a Google Poppins font for a modern appearance and links to an external style.css file for styling. CSS Grid and Flexbox can be used to create a clean, responsive layout that adapts well to different screen sizes. Interactive card effects, spacing, and color styling improve the overall user experience.

Overall, the SpendWise Dashboard demonstrates the use of HTML for webpage structure and CSS for layout, responsiveness, and visual design, providing a foundation for a personal finance or expense tracking application.
